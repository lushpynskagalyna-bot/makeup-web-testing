# Test Plan — E-commerce Website (MAKEUP)

## 1. Introduction
This document describes the test strategy and scope for manual testing of the MAKEUP e-commerce website.

## 2. Objective
To verify that core website functionality works correctly and meets expected user behavior.

## 3. Scope of Testing

Included:
- Homepage
- Search functionality
- Product details page
- Add to cart / Remove from cart
- Checkout process
- Negative scenarios
- Browser console monitoring

Not included:
- Payment gateway integration
- Performance testing
- Security testing

## 4. Test Types
- Functional Testing
- Negative Testing
- UI Testing
- Exploratory Testing

## 5. Test Environment
- Browser: Google Chrome
- Tools: Chrome DevTools, Google Sheets
- OS: Windows

## 6. Entry Criteria
- Website доступний
- Основний функціонал працює

## 7. Exit Criteria
- All planned test cases executed
- Critical defects documented

## 8. Risks
- Website changes during testing
- Environment instability
